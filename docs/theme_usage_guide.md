# 模板使用说明文档

## 1. 模板概览 (Theme Overview)

### 模板名称

**Personal Blog** (个人博客模板)

### 模板描述

Personal Blog 是一个专为个人内容创作者设计的现代化博客模板。该模板基于 Daisy UI 和 Tailwind CSS 构建，提供了优雅、简洁的界面设计，支持深色模式和多语言切换。模板内置了完整的博客文章管理、项目案例展示、标签分类、全文搜索等功能，让用户可以快速搭建个人博客、作品集、简历网站或小型内容站点。

### 适用场景

本模板适用于以下业务场景：

1. **个人博客**：创建个人技术博客、生活随笔、学习笔记等内容站点
2. **作品展示**：展示个人项目案例、设计作品、摄影作品等
3. **个人简历**：构建在线个人简历网站，展示个人技能、工作经历和项目经验
4. **数字相册**：创建个人照片、艺术作品等视觉内容的展示站点
5. **小型内容站点**：适用于需要内容管理功能的小型网站，如知识分享、经验总结等

### 主要特性

1. **双内容类型支持**
   - 博客文章：支持文章发布、分类展示、标签管理
   - 项目案例：支持项目展示、描述、封面图片配置

2. **完整的页面模板系统**
   - 首页模板（index.liquid）：包含 Hero 区域、项目展示、博客文章列表、搜索表单
   - 文章页面模板（page.liquid）：支持博客文章、项目案例、其他类型的内容展示
   - 搜索页面模板（search.liquid）：提供全文搜索功能，支持关键词和标签搜索
   - 标签页面模板（tag.liquid）：按标签分类展示相关内容

3. **响应式设计**
   - 基于 Tailwind CSS 构建，完美适配桌面端、平板和移动设备
   - 优雅的渐变背景和现代化 UI 设计

4. **深色模式支持**
   - 内置深色模式，提供更好的阅读体验
   - 支持主题颜色自定义配置

5. **多语言支持**
   - 内置支持中文简体、English、Germany、France 四种语言
   - 可扩展更多语言支持

6. **灵活的配置选项**
   - 首页 Hero 区域：可配置头像、站点口号、站点介绍
   - 页头页尾导航：支持自定义 HTML 菜单
   - Head HTML 配置：支持自定义 Meta 标签和头部代码
   - 功能定制块：提供三个自定义 HTML 区域

7. **社交链接集成**
   - 支持 Twitter、Facebook、LinkedIn、YouTube 等社交平台链接配置

8. **搜索功能**
   - 全文搜索支持
   - 标签筛选功能
   - 热门搜索关键词配置

9. **标签系统**
   - 支持多标签管理
   - 标签颜色自定义
   - 标签页面展示

10. **内容管理**
    - 富文本编辑器支持
    - 图片上传和管理
    - 页面描述和封面图配置
