# Baklib CMS Personal Site 模板

Personal Site 个人博客模板，适用于创建个人博客、个人简历、相册、小型内容站点等。

模板信息：

- theme_name: personal_blog
- theme_scope: cms
- theme_author: Baklib
- theme_documentation_url: https://help.baklib.cn/themes/personal-blog
- theme_support_url: https://help.baklib.cn/themes/personal-blog/settings
- Demo演示： https://demo-personal-blog.uibak.com/

模板特点：

- 个人博客
- 个人简历
- 数字相册
- 作品展示
- 小型站点
